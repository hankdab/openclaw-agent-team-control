# OpenClaw Agent Team Control v1.0.0

First public desktop release.

## Highlights

- Native macOS desktop app for OpenClaw operator workflows
- Direct agent conversation view with model switch, attachments, history, and context
- Swarm management dashboard for topology, tasks, nodes, and events
- One-click local OpenClaw bootstrap helper
- Real-data mode with mock fallback for local development and demos

## Included artifacts

- `OpenClaw Agent Team Control-1.0.0-arm64.dmg`
- `OpenClaw Agent Team Control-1.0.0-arm64.zip`
- `OpenClaw Agent Team Control.app`

## Notes

- This build targets Apple Silicon macOS
- The app is currently unsigned; on first launch, use right click -> `Open` if macOS blocks it
- Local OpenClaw availability determines whether the app uses live or mock data
